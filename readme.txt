== Description ==
AI Bulk Post Generator automatically generates multiple WordPress posts with AI-generated content, keywords, and images. Perfect for content automation and SEO optimization.

== Installation ==
1. Upload the `ai-bulk-post-generator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'AI Bulk Posts' in the admin panel to start generating posts.
